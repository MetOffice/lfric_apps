.. _pc2:

The PC2 cloud-scheme
====================

Here we describe the PC2 ("Prognostic Cloud 2") scheme.
As implied by the name, this scheme uses prognostic variables for
the sub-grid cloud-fractions (both liquid and ice)
and condensed water mixing-ratios.
